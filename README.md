An appy on Flask.

# How to operate

Run the commands below on `appy` root path:

## Prepare
  
  - What: Run the command below to prepare environment for `Flask` and `poetry` usage:
  
  ```
  make prepare
  ```

## Run

  - What: Run the command below:

  ```
  make start
  ```

## Lint

  - What: Run the command below:

   ```
   make lint
   ```

## Test

  - What: Run the command

  ```
  make test
  ```
  
## Test watch

  - What: Run the command

  ```
  make watch
  ```

## Coverage

  - What: Run the command below:

  ```
  make coverage
  ```
